# Heart Disease Prediction using Machine Learning
A Machine Learning project that predicts the likelihood of heart disease based on patient medical attributes. This project uses the Heart Disease Dataset from Kaggle and applies Logistic Regression for binary classification.

## Project Overview

This project predicts whether a person is likely to have heart disease using Machine Learning techniques. The model is trained using medical data such as age, cholesterol level, blood pressure, heart rate, and other health parameters.

The project uses Logistic Regression for classification and provides accurate predictions based on patient data.

---

##  Dataset

Dataset Link:  
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

The dataset contains medical attributes including:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- ECG Results
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Number of Major Vessels
- Thalassemia
- Target Variable
